AdventureJab is an action-adventure video game with elements of platform games. It is designed specifically for split-screen cooperative multiplayer, which means it must be played with another player via local or online play.

Funny

The game features a large number of mechanics from various video game genres These game mechanics are related to the level's story and theme.

Competitive

For example, in one level, Cody gains the ability to rewind time, while May can replicate herself. Players must cooperate with each other and use these abilities to progress. The game also features a large number of mini-games.

multiplayer

The video game does not have a single player option; it can only be played in online or local split-screen co-op between two players.
